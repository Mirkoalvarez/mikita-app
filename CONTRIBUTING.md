# Contribuir a Mikita Nail Bar

¡Gracias por tu interés en contribuir! 🤎

## Cómo contribuir

### Reportar Bugs

1. Verificá que el bug no haya sido reportado previamente en [Issues](../../issues)
2. Creá un nuevo Issue con:
   - Descripción clara del problema
   - Pasos para reproducirlo
   - Comportamiento esperado vs. actual
   - Screenshots si aplica
   - Dispositivo y navegador utilizado

### Proponer mejoras

1. Abrí un Issue describiendo la mejora
2. Explicá el caso de uso y el beneficio
3. Esperá feedback antes de empezar a codear

### Pull Requests

1. Hacé fork del repositorio
2. Creá una rama para tu feature: `git checkout -b feature/mi-mejora`
3. Commiteá tus cambios: `git commit -m "feat: descripción de la mejora"`
4. Pusheá la rama: `git push origin feature/mi-mejora`
5. Abrí un Pull Request con descripción detallada

## Convenciones de código

- **Componentes**: PascalCase (ej: `NailDesigner.jsx`)
- **Utilities**: camelCase (ej: `formatters.js`)
- **CSS**: Tailwind v4 con tokens definidos en `globals.css`
- **Commits**: Seguir [Conventional Commits](https://www.conventionalcommits.org/)
  - `feat:` nueva funcionalidad
  - `fix:` corrección de bug
  - `docs:` cambios en documentación
  - `style:` cambios de formato/estilo
  - `refactor:` refactorización de código

## Entorno de desarrollo

```bash
npm install
npm run dev
```

El servidor corre en `http://localhost:3000`.
